# Abi's Database model challenge submission

TechStack:
- Node JS - TypeScript
- Mocha and Chai
- Test coverage via nyc

### Installing the packages:

```sh
cd database_table_dependancies
npm install
```

### Run:

```sh
npx ts-node app.ts
```

### Testing: 

```sh
npm test
```

### Test coverage: 

```sh
npm run coverage
```
